🐺 X Following List Exporter

How to Use:

  1. Drag the blue button below to your browser's bookmarks bar
  2. Go to https://twitter.com/YOUR_USERNAME/following
  3. Click the bookmarklet while on the page
  4. Wait for the scroll to complete (it auto-scrolls through your list)
  5. A .txt file will download with all usernames

📥 Bookmarklet:

🔖 Export X Following List

⚠️ Notes:

  • You must be logged in to X/Twitter
  • The script will auto-scroll through your entire following list
  • For large lists (1000+), it may take 1-2 minutes
  • X may rate-limit if you scroll too fast (the script has built-in delays)

Alternative: Copy the JavaScript

If drag-drop doesn't work, create a new bookmark manually:

  1. Right-click bookmarks bar → Add Page
  2. Name: X Following Export
  3. URL: Paste the JavaScript from scripts/x-following-export.bookmarklet.js
