📸 X Media Scraper

Downloads all images/videos from a user's profile while you browse.

How to Use:

  1. Drag the blue button to your bookmarks bar
  2. Go to any X profile: https://twitter.com/USERNAME or https://twitter.com/USERNAME/media
  3. Click the bookmarklet - it will auto-scroll and collect media
  4. When done, a ZIP file downloads with all media

⚠️ Important Notes:
  • Downloads a list of URLs (not the actual files)
  • Use a download manager (wget, curl, browser extension) to batch download
  • For large profiles, may take several minutes
  • X may rate-limit if you scroll too fast

📥 Batch Download the URLs:

After getting the URL list, use one of these:

Option A: wget (Linux/Mac)

wget -i username-media-urls.txt -O media/

Option B: curl (Linux/Mac)

cat username-media-urls.txt | xargs -n 1 curl -O

Option C: Browser Extension

Install "Simple Mass Downloader" or "DownThemAll!" and import the URLs
